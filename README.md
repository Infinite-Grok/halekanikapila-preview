# Hale Kanikapila — Pitch Mockups

Static HTML mockups proposing a redesign of halekanikapila.com.

**Live preview:** https://infinite-grok.github.io/halekanikapila-preview/

These mockups are presentation-only. They are not affiliated with the
business and use publicly available imagery from halekanikapila.com for
demonstration purposes.

## Pages
- `index.html` — Homepage with hero "Tonight / This Week" widget
- `events.html` — Weekly lineup with vibe descriptors
- `owner-console.html` — Mockup of a stripped-down owner dashboard
